Etiquetas de Bloque

hr

address

Esta etiqueta resalta las direcciones:
7a. Calle 3-40 zona 1

blockquote

Esta etiqueta establece una identacion para citas:
"El respeto al derecho ajeno es la paz."
Benito Juarez

details

Esta etiqueta se usa en conjunto con la etiqueta summary
